## Task 03: Set up pay-as-you-go billing

## Description
You'll create a billing plan named CUAgentsBillingPlan in Power Platform admin center, link it to your Azure subscription and the RGforCUAgentsResouces resource group, and attach it to the EnvironmentForCUAgents environment. The plan must include the Dataverse, Copilot Studio, and Windows 365 for Agents meters.

## Success criteria
- You created a billing plan named CUAgentsBillingPlan linked to your Azure subscription and resource group.
- You selected all three required meters: Dataverse, Copilot Studio, and Windows 365 for Agents.
- 	You confirmed the EnvironmentForCUAgents environment appears in the Target environments section of the billing plan.


Configure a pay-as-you-go billing plan in Power Platform admin center before you deploy a cloud PC pool.

In this task, you will create the billing plan.

{: .warning } 
> You must perform this task in Power Platform admin center. Creating billing for Microsoft 365 Copilot chat will fulfill the requirements.


1. Open an InPrivate browser session in **Microsoft Edge** and go to `https://admin.powerplatform.microsoft.com/`. The Power Platform admin center **Home** page displays.

1. Sign in with your tenant administrative credentials.

1. In the left pane, expand **Licensing**. 

	![vswr2udj.png](../../media/vswr2udj.png)

1. In the **Licensing** pane, select **Billing Plans**.

	![k8a9rl5y.png](../../media/k8a9rl5y.png)

1. On the **Billing plans** page, on the command bar, select **+ New billing plan**.

	![uytvmcoc.png](../../media/uytvmcoc.png)

1. In the **New billing plan** pane, select **Azure subscription**.
	
	![29ptayul.png](../../media/29ptayul.png)

1. Configure the fields by using the information in the following table:

	| Field | Value |
	|:---------|:---------|
	| Name   | `CUAgentsBillingPlan`   |
	| Azure subscription   | Select the Azure subscription that you would like to bill.   |
	| Resource group   | **RGforCUAgentsResouces**   |

	![5ibrjq5g.png](../../media/5ibrjq5g.png)

1. In the **Meter** field, select the following meters and then select **Next**.

	- Dataverse
    - Copilot Studio
    - Windows 365 for Agents
    
	![v8z56oja.png](../../media/v8z56oja.png)
	![rhfjrm1b.png](../../media/rhfjrm1b.png)

1. In the **Select environment** pane, in the **Region** field, select **United States**.

	![5rpn62vn.png](../../media/5rpn62vn.png)

1. In the list of environments, select **EnvironmentForCUAgents** and then select **Save**.

	![chn70jlp.png](../../media/chn70jlp.png)
	![uvceenkp.png](../../media/uvceenkp.png)

	{: .warning }
	> During testing, on multiple occasions we observed that the environment did not attach to the billing plan after saving. To check this, select the billing plan. Then, on the command bar, select **Edit**.
	>
	![7o1s2x5u.png](../../media/7o1s2x5u.png)
	>
	If you do not see the **EnvironmentForCUAgents** environment listed in the **Target environments** section, select **Add or remove environments**.
	>
	![5g2j7cp4.png](../../media/5g2j7cp4.png)
	>
	Select your environment again and then select **Update environments**.
	>
	![rq2edha4.png](../../media/rq2edha4.png)
	![gide5thj.png](../../media/gide5thj.png)
	>
	Finally, select **Save**.
